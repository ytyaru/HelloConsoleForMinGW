# このソフトウェアについて #

HelloConsoleForMinGWは私個人が学習目的で作成したソフトウェアである。

MinGWのgccでHelloWorldした。（ConsoleApplication版）

# 開発環境 #

* Windows XP Pro SP3 32bit
* MinGW
    * msys 1.0
    * gcc 4.6.2
